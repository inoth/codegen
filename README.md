# codegen
```shell
PS C:> .\codegen.exe --help
Usage of C:codegen.exe:
  -db string
        database connect string 数据库连接串，非空
  -f int
        defualt:0; 是否格式化处理数据命名格式，1：处理，0：不处理
  -pn string
        project name 项目名称 (default "DefualtProject")
  -tbls string
        tables; 'tableA,tableB,tableC' 需要单独生成表名称; 不传默认生成全表
```
